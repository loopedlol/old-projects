# Flask Math Calculator

## Description
A Flask practice app with routes and buttons for factorial, Fibonacci, and average calculations.

## Original Context
This was an older learning project.

## How to Run
Install dependencies, then run the app:

```sh
python3 -m pip install -r requirements.txt
python3 app.py
```

Open the local Flask URL shown in the terminal, usually `http://127.0.0.1:5000/`.
