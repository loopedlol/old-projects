# Mandelbrot Viewer

## Description
A Python Mandelbrot set image generator using Pillow and Matplotlib.

## Original Context
This was an older learning project.

## How to Run
Run from the project folder:

```sh
python3 -m pip install -r requirements.txt
python3 mandelbrotV3.py
```
